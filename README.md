# ContactSphere
**A User friendly, highly interactive Contacts App**

### App Awesome Functionalities
 - Contacts are saved in cloud, so it's for a long run.
 - Only you can see your saved contacts, and your information won't be shared no matter the case.
 - Only Authenticated Users have access to using the App.
 - You can Hide, Trash, Delete and Star Contacts.
